SnapToPerfectLine(IntVector2, IntVector2, int)

Returns the closest (in l1 distance) point to movablePoint that forms a perfect Line to fixedPoint.

public static IntVector2 SnapToPerfectLine(IntVector2 fixedPoint, IntVector2 movablePoint, int maxDistance = 100)

Parameters

fixedPoint IntVector2
movablePoint IntVector2
maxDistance int

The largest l1 distance from movablePoint to search for a point to snap to.

Returns

IntVector2

Remarks

Note such a point may not be unique. This method breaks ties in such a way that applying an isometry to fixedPoint and movablePoint (the same one to both) applies the isometry to the snapped point.

Exceptions

ArgumentOutOfRangeException

maxDistance is negative.

InvalidOperationException

There is no point to snap to within maxDistance of movablePoint.

SnapToSquare(IntVector2, IntVector2)

Either changes movablePoint's x coord or changes its y coord so that the rectangle it forms with fixedPoint is a square.

public static IntVector2 SnapToSquare(IntVector2 fixedPoint, IntVector2 movablePoint)

Parameters

fixedPoint IntVector2
movablePoint IntVector2

Returns

IntVector2

Remarks

Chooses the largest such square that preserves the sign of each component of movablePoint - fixedPoint. If fixedPoint and movablePoint have the same x coord (but different y), the square will be made by increasing the x coord; if fixedPoint and movablePoint have the same y coord (but different x), the square will be made by increasing the y coord.
