Remarks

The picker gets the hue as the x coord on the HSLHueSaturationBox (in the range [0, 1]), the saturation as the y coord on the HSLHueSaturationBox (in the range [0, 1]), and the lightness as the y coord on the HSLLightnessSlider (in the range [0, 1]).

For the display, the shader treats the HSL as being in gamma space. Hence, for the picked colour to match the colour on the display, the picked colour should be treated as in gamma space.
